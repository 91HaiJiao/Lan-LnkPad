# LAN InkPad Enhanced 0.3

增强版在稳定版的全部绘图、坐标范围、浏览器网页覆盖和工具功能之上，增加当前标签页实时画面传输、双模式缩放和平滑笔迹。

## 与稳定版并存

- 稳定版目录：`D:\codex work\lan-inkpad`
- 增强版目录：`D:\codex work\lan-inkpad-enhanced`
- 稳定版服务端口：`8787`
- 增强版服务端口：`8788`
- 增强版 Android 包名：`com.laninkpad.enhanced`

两个 Windows 服务和两个 APK 可以同时存在。增强版开发不会覆盖稳定版。

## 安装增强版扩展

1. Chrome 打开 `chrome://extensions/`，Edge 打开 `edge://extensions/`。
2. 开启开发者模式。
3. 如果同时保留旧扩展，建议先把旧扩展关闭，避免点击时混淆。
4. 点击“加载已解压的扩展”，选择增强版的 `extension` 文件夹。
5. 固定名为“LAN InkPad 增强版”的扩展。

## 启动和配对

1. 双击增强版目录内的 `start-windows.cmd`。
2. 扩展服务地址填写 `127.0.0.1:8788`。
3. 手机增强版 APK 填写电脑 WLAN 地址，例如 `192.168.1.103:8788`。
4. 手机与扩展填写相同的 6 位配对码。
5. 在普通网页启用“在当前网页显示标注层”。

## 实时画面模式

1. 手机和扩展先完成配对，扩展状态必须显示“手机已连接”。
2. 打开要标注的网页。
3. 扩展中选择清晰度：
   - 省电：960×540，15fps，约 0.8Mbps。
   - 流畅：1280×720，20fps，约 1.5Mbps，低配电脑推荐。
   - 高清：1920×1080，24fps，约 3Mbps。
4. 点击“启动当前标签页实时画面”。
5. 手机切换到“实时画面”。
6. 单指绘图；双指捏合放大并拖动；右下角按钮也可缩放和复位。

画面使用 WebRTC 在局域网内点对点传输，不上传云端，不传音频。扩展关闭、标签页关闭或手机断开后会自动释放采集资源。

## 坐标板模式

坐标板模式保留稳定版的严格 ROI 映射，也支持 1～6 倍双指缩放和平移。缩放只改变手机观察和操作精度，发送到电脑的仍是正确的 ROI 归一化坐标。

## 性能建议

- 低配电脑优先使用“流畅 720p / 20帧”或“省电 540p / 15帧”。
- 只在需要的标签页启用标注层。
- 使用结束后点击“停止实时画面”。
- 不要同时运行多个屏幕录制、视频会议或高分辨率直播程序。

## 浏览器限制

实时画面捕获的是当前普通网页标签页，不包括浏览器地址栏、标签栏和系统窗口。`chrome://`、`edge://`、扩展商店和其他受保护页面不能注入或捕获。部分内置 PDF 查看器仍可能受浏览器限制。
