# Chrome / Edge 浏览器扩展

扩展目录为 `extension/`，使用 Manifest V3，不需要打包即可在开发者模式下加载。

## 安装到 Chrome

1. Windows 双击 `start-windows.cmd`，保持命令窗口运行。
2. Chrome 地址栏打开 `chrome://extensions/`。
3. 打开右上角“开发者模式”。
4. 点击“加载已解压的扩展程序”。
5. 选择本项目的 `extension` 文件夹。
6. 建议把“LAN InkPad 增强版”固定到浏览器工具栏。

## 安装到 Edge

1. Windows 双击 `start-windows.cmd`。
2. Edge 地址栏打开 `edge://extensions/`。
3. 打开“开发人员模式”。
4. 点击“加载解压缩的扩展”。
5. 选择 `extension` 文件夹。

## 配对

1. 点击浏览器工具栏里的 LAN InkPad 图标。
2. 服务地址填写 `127.0.0.1:8788`。
3. 记录或更换 6 位配对码。
4. 手机增强版 APK 填写电脑的局域网地址，例如 `192.168.1.8:8788`，配对码必须相同。
5. 在需要标注的网页上打开扩展弹窗，启用“在当前网页显示标注层”。

扩展右上角状态条会显示鼠标当前的网页视口逻辑坐标以及 ROI 坐标范围。手机端可在“坐标范围”中输入这些数值，进行局部精确映射。

## 工作方式

- 扩展后台只有一条 WebSocket 连接，以 `desktop` 角色进入房间。
- 笔迹只发送到当前激活且已启用标注的标签页。
- 每个标签页拥有独立 Canvas 和撤销历史。
- 标注层固定在浏览器视口上；网页滚动时，笔迹保持在屏幕原位置。
- 页面刷新或关闭标签页会清除该标签页的笔迹。
- Canvas 完全鼠标穿透，因此不影响网页点击、滚动和输入。

## 无法覆盖的页面

浏览器安全机制禁止扩展注入以下页面：

- `chrome://` 与 `edge://` 内部页面
- Chrome 网上应用店、Edge 加载项商店
- 其他扩展的页面
- 浏览器地址栏、标签栏、工具栏和开发者工具
- Chrome/Edge 内置 PDF 查看器在部分版本中也属于受限扩展页面

普通 `http://` 和 `https://` 网页均可覆盖。若 PDF 被内置查看器限制，可继续用原项目网页内打开 PDF，或后续加入扩展自带 PDF 阅读器。

## 更新扩展

修改 `extension/` 文件后，在扩展管理页面点击该扩展的“重新加载”，然后刷新需要标注的网页。
