# LAN InkPad 增强版使用教程

## 一、运行条件

- Windows 10 或 Windows 11 电脑
- Android 手机
- Chrome 或 Microsoft Edge 116 及以上版本
- 电脑和手机连接同一个 Wi-Fi
- 电脑安装 Node.js 20 或更新版本：[https://nodejs.org/](https://nodejs.org/)

## 二、电脑端启动服务

1. 解压收到的完整项目压缩包。
2. 进入 `LAN-InkPad-Enhanced-0.3.1-Share` 文件夹。
3. 双击 `start-windows.cmd`。
4. Windows 防火墙第一次询问时，允许“专用网络”。
5. 浏览器会打开本机 IP 查询页面。
6. 找到名称为 `WLAN` 或 `Wi-Fi` 的地址，例如 `192.168.1.20:8788`。
7. 保持 CMD 窗口开启；关闭窗口会停止服务。

如果电脑尚未安装依赖，启动程序会自动执行 `npm install`。

## 三、安装增强版浏览器扩展

### Edge

1. 地址栏打开 `edge://extensions/`。
2. 开启“开发人员模式”。
3. 点击“加载解压缩的扩展”。
4. 选择 `release/LAN-InkPad-Enhanced-Extension-0.3.1-unpacked` 文件夹。
5. 把“LAN InkPad 增强版”固定到浏览器工具栏。

### Chrome

1. 地址栏打开 `chrome://extensions/`。
2. 开启“开发者模式”。
3. 点击“加载已解压的扩展程序”。
4. 选择 `release/LAN-InkPad-Enhanced-Extension-0.3.1-unpacked` 文件夹。

不要直接选择 ZIP 文件，也不要选择 `release` 上级目录。

## 四、安装手机 APK

APK 位于：

```text
release/LAN-InkPad-Enhanced-0.3.0-debug.apk
```

把 APK 发送到 Android 手机并安装。如果系统阻止安装，需要临时允许文件管理器或浏览器“安装未知应用”。

增强版包名为 `com.laninkpad.enhanced`，不会覆盖普通版 LAN InkPad。

## 五、电脑和手机配对

1. 打开任意普通网页。
2. 点击浏览器工具栏中的“LAN InkPad 增强版”。
3. 扩展的服务地址填写：

```text
127.0.0.1:8788
```

4. 点击“换一个”生成 6 位配对码，然后点击“保存并连接”。
5. 手机打开 LAN InkPad Enhanced，电脑地址填写 IP 查询页面中的 WLAN 地址，例如：

```text
192.168.1.20:8788
```

6. 手机填写和扩展完全相同的 6 位配对码，点击连接。
7. 扩展显示“手机已连接”即表示配对成功。

注意：每台电脑的 WLAN IP 不同，不要照抄示例地址。

## 六、实时画面模式

1. 打开需要标注的普通网页。
2. 点击增强版扩展。
3. 选择“流畅 720p / 20帧（推荐）”。低配电脑可选择“省电 540p / 15帧”。
4. 点击“启动当前标签页实时画面”。扩展会自动启用当前标签页标注层。
5. 手机切换到“实时画面”。
6. 手机单指绘图，笔迹会显示在电脑当前标签页。
7. 手机双指捏合可放大或缩小，双指拖动可平移画面。
8. 使用结束后点击扩展中的“停止实时画面”。

实时画面只传输当前浏览器标签页，不包括浏览器地址栏、标签栏或整个 Windows 桌面。

## 七、坐标板模式

手机切换到“坐标板”，即可使用不传输画面的低延迟坐标模式。该模式同样支持双指缩放和平移，以及精确 ROI 坐标范围。

## 八、绘图工具

- 普通笔：压感和平滑线条
- 荧光笔：半透明标注
- 激光笔：自动淡出
- 橡皮：擦除永久笔迹
- 颜色与粗细调节
- 撤销与清屏

## 九、常见问题

### 扩展没有显示

进入 `edge://extensions/` 或 `chrome://extensions/`，确认增强版扩展已启用。重新加载扩展后，需要刷新目标网页。

### 手机连接失败

- 确认电脑和手机在同一个 Wi-Fi。
- 使用 WLAN 地址，不要使用 VMware、VPN、`localhost` 或 `127.0.0.1`。
- 确认手机使用端口 `8788`。
- 确认两端配对码相同。
- 检查 Windows 防火墙是否允许 Node.js 使用专用网络。

### 画面提示存在旧采集流

在扩展管理页面点击“重新加载”，刷新目标网页后重试。关闭其他录屏、视频会议或屏幕共享程序。

### 实时画面卡顿

切换到“省电 540p / 15帧”，关闭不必要的标签页和录屏软件，并尽量使用 5GHz Wi-Fi。

### 哪些页面不能使用

`chrome://`、`edge://`、浏览器扩展商店、其他扩展页面以及浏览器工具栏属于受保护区域，不能注入标注层。
